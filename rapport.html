Le HTML

Le langage HTML (HyperText Markup Language) permet de placer des elements sur une page web. Il permet au navigateur de savoir ce qu'il faut afficher et dans quel ordre les afficher. Le code HTML n'est pas un langage de programmation mais un langage compose de balises definissant le contenu compose d'une serie d'elements. Un Voici a quoi servent certaines balise de code HTML:

  • < !DOCTYPE html >: dit au navigateur que le document (= la page) est code en HTML
  • < head >: permet de definir certaines proprietes de la page qui ne s'affichent pas sur celle-ci. Voici quelques elements pouvant etre trouves dans le head:
    • < meta charset="utf-8" >: Definit l'encodage a utiliser pour les characteres
    • < title >: Definit ce qui se trouve en titre sur l'onglet
    • < link rel="stylesheet" href="style.css" >: Definit quel fichier css (ici style.css) utiliser pour definir les styles
  • < div >: permet de mettre plusieurs elements dans un groupe (je crois)
  • < h1 a h6 >: Permet de definir des elements comme des titres de plus ou moins grande importance (h1= plus gros titre et h6= plus petit)
  • < p >: Designe du texte comme un paragraphe
  • < a href="url vise" >: Permet de definir un element comme etant un lien vers un url donne dans href
  • < img src="Image" >: Permet d'ajouter une image dont la source est definie apres src
  • < button >: definit un element comme un bouton
  • < iframe >: permet d'integrer un document HTML dans un document HTML
  • < html >: definit la partie du document code en html
  • < body >: definit la partie du document qui compose le corps du texte
  • < video src="Video" type="type" >: permet d'integrer une video dans le site web. La source de la video est definie apres src et le format de la video apres type
  • class="" permet de definir des elements comme appartenant a une classe (utile surtout pour le css)

Le CSS

Le CSS ou Cascading Style Sheets est le code qui permet de definir a quoi vont ressembler les differentes choses de la page que nous avons code en HTML. Avec le CSS on peut definir les polices, les positions, les arrieres plans, les tailles, etc. de la page. On peut aussi faire changer des choses lorsque le curseur passe dessus et meme creer des animations basiques (comme le changement de couleur sur la page Accueuil). Lorsqu'on veut appliquer des effets a un type d'elements on ecrit:

balise (sans les <>) {code css}

ou

.classe {code css}

Quelques elements CSS:

  • background: permet de definir l'arriere plan d'un element
  • color: permet de definir la couleur de l'ecriture (background-color permet de definir la couleur d'un arriere plan)
  • text-align: permet de definir l'alignement du texte
  • margin: definit les marges d'un element
  • width/ height: definit la hauteur et la largeur d'un element
  • right/ top/ left/ bottom: definit le placement d'un element
  • display: montre ou cache des elements
  • float: permet de faire se deplacer de elements vers un cote de l'ecran
  • et beaucoup d'autres

Conclusion

Le HTML et le CSS permettent de creer des pages web. Ca prend beaucoup de temps si on veut faire une belle page avec des elements plus ou moins developpes (les menus defilants restent un peu de la magie pour moi :)). Je trouve le HTML/CSS moins interessant et moins amusant que la programmation avec python et du coup je pense que le java sera interessant a etudier...
